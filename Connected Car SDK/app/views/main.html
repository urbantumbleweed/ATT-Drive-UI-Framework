Hello World!

This AT&T Drive Automotive App development UI Kit is built to enable you -the developer, to effortlessly create apps to be deployed on AT&T Drive Automotive Services Delivery Platform.

At this time this is only an UI toolkit, with a goal to by 2015 include Drive Platform APIS and become an all-around SDK.

Please select the component on the left to learn more about it. To create fresh Seed App with required libraries use $ grunt seed then look for myFirstApp folder in root.

Voice is an app concept

AT&T Drive is a cloud based automotive services delivery platform and the apps have the ability to be fully voice enabled as part of AT&T initiative to prevent distracted driving. This document, including the design guidelines and the Information Architecture stencils are simplified to enable creation of automotive apps that are effortless and distraction-free

Voice as an app flow:

Common source code, less clutter, faster apps, better experience

The intent of this toolkit is to help automotive manufacturers maintain their brand presence in the in-car digital experience. As app developer build apps from the same framework of libraries and APIs, the result is structured, seamless, easy brand-able set of applications, and as AT&T Drive offers theming per vehicle make/model/trim, reusing common UI libraries and app source-code is crucial.
