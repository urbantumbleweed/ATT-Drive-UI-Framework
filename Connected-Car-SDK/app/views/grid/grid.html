Grid system

The SDK supports a flexbox-based grid system. It's a great tool for defining your app's layout, and it works well in conjuction with the att-content directive.

Usage:

Note

If you use the att-content directive, you don't need to specify the "att-grid" wrapper, since it's already included.

Basic grids

Nest as many rows and columns as you need. Simply add columns you want in a row, and they'll evenly take up the available space.

Individual sizing

You can explicitly state the width of a column if for example you'd want specific columns to be larger than the others in the same row. By default each column will evenly take up the available area.
